# Face Recognition Attendance System

## Project Overview

This project is a Machine Learning based Face Recognition Attendance System developed using Python and Google Colab.

The system recognizes a person's face from an uploaded image and automatically generates an attendance record with the student's ID, name, date, time, and attendance status.

---

## Features

- Face Detection
- Face Recognition
- Student Identification
- Automatic Attendance Generation
- Attendance CSV Export
- Output Image Generation

---

## Technologies Used

- Python
- Google Colab
- OpenCV
- face_recognition
- NumPy
- Pandas
- Matplotlib

---

## How to Run

1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Upload the Known_Faces folder.
4. Encode the known faces.
5. Upload a test image.
6. Run face recognition.
7. Generate attendance.

---

## Output

The system generates:

- Attendance CSV
- Recognized Output Image
